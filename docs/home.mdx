# Welcome to the EssentialsX Wiki!

EssentialsX is a Bukkit plugin designed to enhance your Minecraft server by adding over 130 essential commands and features. This page serves as your go-to guide for receiving support, reading documentation, and reporting bugs. Whether you're a new user or an experienced administrator, we've compiled all the necessary resources to help you make the most of EssentialsX.

### Reading the Documentation
Our comprehensive documentation aims to cover all aspects of the plugin, from basic setup to advanced configuration. To find detailed information about each command, feature, and setting, select a page from the sidebar on the left. The documentation is divided into several sections to help you easily navigate through different topics.

### How to Get Support
If you encounter any issues or have questions about EssentialsX, feel free to join the [Discord Server](https://discord.gg/h8CnPSw), where we will be able to effectively guide you in the right direction to proceed.

## Reporting Bugs
Encountered a bug? Help us improve EssentialsX by reporting it. Here’s how you can report a bug effectively:

* Check Existing Reports: Before submitting a new bug report, please check our Bug Tracker to see if the issue has already been reported. This helps avoid duplicate reports.
* Detailed Reports: When reporting a bug, be as detailed as possible. Include as much information as possible, including a link to a server dump `/ess dump all`, and any relevant error logs or screenshots. This information is crucial for us to understand and correct the issue.

If you are unsure whether something is a bug, or just a misconfiguration, we encourage you to reach out to the Discord Support team!

## Contributing to EssentialsX
EssentialsX is an open-source project, and contributions from the community are welcome! If you’re interested in contributing, whether through coding, documentation, or providing translations, visit the [EssentialsX Development Discord](https://discord.gg/CUN7qVb)

## Stay Updated
Stay up-to-date with the latest developments and updates by following us on GitHub and joining our community on Discord. We regularly post updates and information about upcoming features and enhancements.
